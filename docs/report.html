FedThreat-X Research Report

FedThreat-X enables privacy-preserving federated threat intelligence for multi-cloud cybersecurity by sharing model updates instead of raw surveillance logs.

Abstract Summary

The framework addresses cross-cloud detection challenges caused by sensitive logs, inconsistent security policies, heterogeneous cloud architecture, and non-IID telemetry. It combines threat-aware adaptive aggregation, proximal stability management, and lightweight local MLP classification to train across cloud nodes while preserving confidentiality.
